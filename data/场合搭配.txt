一、日常休闲场合（逛街、散步、聚会、出游）
日常休闲追求舒适、自在、不刻意，面料优先选择棉质、针织、牛仔等柔软材质。
上衣可选择 T 恤、卫衣、针织衫、休闲衬衫，下装搭配牛仔裤、运动裤、休闲裤、半身裙。
经典搭配：白色 T 恤 + 蓝色牛仔裤 + 小白鞋，清爽不出错，适合绝大多数人。
春秋休闲：卫衣 + 束脚运动裤 + 运动鞋，舒适随性，适合外出、散步、短途出行。
夏季休闲：短袖 Polo 衫 + 卡其短裤 + 帆布鞋，简约干净，显精神。
冬季休闲：毛衣 / 针织衫 + 休闲裤 + 羽绒服 / 棉服，保暖又不臃肿。
微胖体型休闲穿搭：上浅下深、宽松上衣 + 直筒裤，遮肉显瘦，不紧绷。
偏瘦体型休闲穿搭：合身版型 + 轻微层次感，避免过于宽松显得没精神。
休闲风颜色以黑白灰、蓝、卡其、米色为主，柔和耐看，不挑场景。
鞋子优先小白鞋、帆布鞋、运动鞋，舒适百搭，符合轻松氛围。
二、职场通勤场合（办公室、商务会谈、正式工作）
职场穿搭强调干练、得体、专业，避免过于暴露、花哨、宽松随意。
男性职场标配：衬衫 + 西裤 / 休闲西裤 + 皮鞋，春秋可加西装外套。
女性职场标配：衬衫 / 针织衫 + 西装裤 / 直筒半身裙 + 小皮鞋 / 低跟鞋。
颜色优先黑白灰、藏青、深咖、米色等低饱和色，显得稳重专业。
夏季职场：短袖衬衫 + 九分西裤，清爽利落，不随意。
冬季职场：毛衣 + 衬衫叠穿 + 西装外套 + 毛呢西裤，保暖又正式。
避免夸张图案、大 logo、亮片、透视装，不符合职场严肃氛围。
裤子优先直筒、微阔版型，不紧身、不拖沓，显得干练精神。
裙子建议在膝盖附近，过短不够正式，过长显得拖沓。
鞋子以低跟、粗跟、乐福鞋为主，舒适且符合职场形象。
三、校园学生场合（上课、图书馆、食堂、校园活动）
校园风主打青春、简约、干净、舒适，不用过度打扮。
经典搭配：连帽卫衣 + 牛仔裤 + 小白鞋，青春感强，适合学生。
衬衫 + 针织马甲 + 休闲裤，学院风明显，显乖巧文静。
女生常见：百褶裙 + 毛衣 / T 恤 + 中筒袜 + 小皮鞋，日系学院风。
男生常见：棒球服 / 夹克 + 牛仔裤 + 运动鞋，阳光少年感。
上课穿搭以宽松舒适为主，久坐不勒，方便活动。
颜色以浅蓝、白色、米色、灰色、浅咖为主，显年轻干净。
避免过于成熟、暴露、夸张的穿搭，与校园氛围不符。
书包、帆布包、棒球帽是校园风常见配饰，提升整体感。
下雨天可搭配冲锋衣、休闲外套，实用又好看。
四、约会 / 心动场合（约会、逛街、看电影、吃饭）
约会穿搭重点：温柔、显气质、显身材优势、不刻意做作。
女生约会首选：连衣裙、针织衫 + 半身裙、衬衫 + A 字裙，温柔显气质。
男生约会首选：简约 Polo 衫、合身 T 恤、针织衫 + 休闲裤 / 牛仔裤，干净清爽。
颜色优先柔和色系：粉色、米色、浅蓝、浅紫、奶白、咖色，显温柔。
版型以合身为主，不过紧不松垮，突出身材比例。
冬季约会：毛衣 + 大衣 / 毛呢外套，温柔大气，显氛围感。
夏季约会：碎花裙、纯色吊带 + 薄开衫，清爽又有女人味。
男生避免花里胡哨图案，简单纯色最显好感。
鞋子建议小皮鞋、乐福鞋、小白鞋，干净整洁很加分。
可轻微配饰点缀，如项链、手链、围巾，提升精致度但不夸张。
五、正式宴会 / 晚宴场合（婚礼、年会、晚宴、重要仪式）
正式场合要求端庄、大气、高级，面料优先西装料、丝绒、雪纺、礼服面料。
男性标配：深色西装套装 + 衬衫 + 领带 + 皮鞋，整体统一协调。
女性标配：礼服裙、连衣裙、西装套裙，颜色以黑、酒红、墨绿、藏青、金色为主。
避免休闲面料如牛仔、运动面料、卫衣，显得不尊重场合。
婚礼作为宾客：避免全身纯白或纯红，以免与新人冲突。
年会穿搭可稍微亮眼，如小面积亮片、丝绒材质，提升气场。
正式场合全身颜色不超过两种，显得高级不乱。
鞋子必须正式，男士皮鞋，女士中高跟 / 细跟鞋。
发型整洁、配饰简约，整体端庄大气。
冬季正式场合可搭配大衣、毛呢外套，进场后脱下，保持内部精致。
六、运动健身场合（跑步、健身、打球、瑜伽）
运动穿搭核心：透气、吸汗、弹力好、不束缚动作。
上衣选择速干 T 恤、运动背心、健身短袖，排汗不粘身。
下装选择运动短裤、紧身健身裤、束脚运动裤，弹力舒适。
跑步穿搭：速干衣 + 运动短裤 + 跑鞋，轻便透气。
瑜伽穿搭：高弹紧身衣 + 瑜伽裤，柔软贴身，不影响伸展。
球类运动：宽松运动套装，活动空间大，不闷汗。
冬季户外运动：加绒运动套装 + 防风外套，保暖不笨重。
颜色鲜艳更显活力，黑、灰、蓝、荧光色都常见。
必须穿运动鞋，保护脚踝，提升运动表现。
避免牛仔裤、紧身牛仔、非弹力面料，限制运动幅度。
七、居家休闲场合（在家、居家办公、取快递、楼下散步）
居家穿搭以极度舒适、宽松、柔软为主，方便活动。
常见搭配：睡衣套装、T 恤 + 家居裤、宽松卫衣 + 休闲裤。
面料优先纯棉、莫代尔、珊瑚绒，柔软亲肤。
冬季居家：珊瑚绒睡衣、法兰绒套装，保暖舒服。
夏季居家：短袖短裤、冰丝睡衣，清爽不闷。
偶尔下楼取快递：可外搭薄外套、拖鞋换帆布鞋，快速出门不尴尬。
居家办公可稍微正式一点，如宽松 T 恤 + 休闲裤，视频会议不突兀。
颜色以浅色系、柔和色为主，让人放松心情。
八、秋冬保暖厚重场合（低温、大风、过冬）
秋冬穿搭核心：保暖、防风、有层次感，不显臃肿。
三层穿搭法：内层吸汗（秋衣 / 薄 T 恤）+ 中层保暖（毛衣 / 针织）+ 外层防风（羽绒服 / 大衣）。
冬季标配：高领毛衣 + 毛呢裤 / 牛仔裤 + 羽绒服 / 棉服 + 靴子。
大衣穿搭：针织衫 + 衬衫 + 休闲裤 + 毛呢大衣，显高显瘦气质强。
腿冷可加秋裤、打底裤，选择修身弹力款，不显腿粗。
颜色偏深：黑、灰、藏青、咖色、墨绿，耐脏又显瘦。
鞋子可选雪地靴、马丁靴、短靴，保暖又百搭。
围巾、帽子、手套不仅保暖，还能提升穿搭层次感。
九、夏季清凉轻薄场合（高温、闷热、出汗多）
夏季穿搭核心：透气、轻薄、防晒、不闷热。
面料优先纯棉、冰丝、亚麻、天丝，吸汗透气。
上衣：短袖 T 恤、衬衫、吊带、背心，宽松更凉快。
下装：短裤、短裙、九分裤、阔腿裤，通风不贴身。
颜色优先浅色：白、浅蓝、浅灰、米色，吸热少更凉快。
避免黑色、深色大面积穿搭，吸热明显，更热。
宽松版型优于紧身，空气流通更强，出汗不粘身。
防晒可搭配薄开衫、防晒衣，不闷又防晒。
十、旅行出游场合（长途出行、拍照、走路多、换场景）
旅行穿搭：舒适耐穿、方便走路、上镜好看、易穿脱。
优先选择耐皱面料，不易皱、不用频繁整理。
裤子选弹力牛仔裤、休闲裤、工装裤，多口袋更实用。
鞋子必须舒适：运动鞋、小白鞋、休闲鞋，长时间走路不累脚。
拍照穿搭：浅色系、纯色、轻微碎花更上镜，不杂乱。
爬山 / 户外：冲锋衣 + 速干裤 + 登山鞋，防风防刮更实用。
十一、复古风穿搭场合（拍照、探店、文艺活动）
复古风以牛仔、卡其、棕色、墨绿色为主，强调年代感。
男生复古：牛仔外套 + 复古 T 恤 + 直筒牛仔裤 + 板鞋。
女生复古：碎花裙、灯芯绒半裙、针织衫 + 牛仔，温柔复古。
配饰可选贝雷帽、帆布包、复古腰带，提升风格。
十二、中性风 / 酷飒穿搭（街头、个性、冷淡风）
中性风颜色以黑白灰、藏青、军绿为主，简约帅气。
常见搭配：西装外套 + T 恤 + 牛仔裤 + 马丁靴。
女生中性风：衬衫 + 阔腿裤 + 皮鞋，利落帅气。
避免过多装饰，线条干净利落，气场更强。
十三、场合搭配避坑（非常适合检索）
职场穿拖鞋、背心、超短裤：不专业，印象差。
约会穿运动装、睡衣风：显得不重视对方。
正式宴会穿牛仔裤、卫衣：显得失礼、不合场合。
运动穿牛仔裤、皮鞋：容易受伤、行动受限。
夏天穿深色紧身衣：闷热、显胖、不透气。
冬天穿过于单薄：影响健康，穿搭也不自然。
同一套衣服通吃所有场合：缺乏礼仪意识，观感不佳。
